Functions

A function $ f : A \rightarrow B $ is a rule which associate each $x$ member of set A which we call domain of $f$ (we also say $f$ is defined on A) to a unique $y$ value in the set B which we call codomain of $f$.

We then write $$ y = f(x) $$

Where we call $f(x)$ the value of $f$ at $x$. The set of values of $f$ as $x$ takes all the values in A is called the range of $f$. The range of the $f$ is a subset of the codomain of $f$.

For some functions, the input might not make sense and/or is not defined. $f(x)= \tfrac{1}{x}$ is one where $x=0$ is impossible. We then say function is not defined at $x = 0$. Since the functions works for all real numbers, we can say that the "function $ f $ is defined by $ f(x) = \tfrac{1}{x}, x \ne 0$" or "function $$f : \mathbb{R}\backslash \{0\} \rightarrow \mathbb{R}$$ is defined by $f(x) = \tfrac{1}{x}$".

Note that $$ \mathbb{R}\backslash \{0\} $$ denote real numbers excluding 0


Even & Odd Functions

$f$ is an even function if $f(x) = f(-x)$

$f$ is an odd function if $f(x) = -f(-x)$

Note that all even functions have a symmetry about the $y$-axis (reflected by the $y$-axis) whereas the odd functions will have a rotational symmetry of order 2. This effectively mean you just have to rotate it by 90 degrees twice to get back to the same graph.

An example of even function is $ f(x) = x^2 $. Both positive and negative $x$ maps to the same positive $y$ value. For the odd function, we can use $ f(x) = x $, where positive $x$ maps to positive $y$ only and vice versa. This satisfies the condition above $f(x) = -f(-x)$.


Periodic Functions

$f$ is a periodic function if $f(x) = f(x+a)$ where $a$ is the period of $f$.

Usually periodic functions are some sort of trignometric functions.


Polynomials Functions

The general polynomial of degree $n$ in $x$ can be written as

$$ y = f(x) = a_nx^n + a_{n-1}x^{n-1} + a_{n-2}x^{n-2} + ... + a_1x^1 + a_0 \nonumber $$

where $a_n, a_{n-2}, a_{n-3} ... a_1, a $ are numbers or coefficients and $a_n \ne 0$. The degree of the polynomial is the highest power of $x$.

A popular polynomial function is the quadratic function, where it is a polynomial of degree 2. Rearranging the polynomial equation can show the position and shape of the graph. $$ f(x) = ax^2 + bx + c \nonumber $$

Dividing all terms by $a$ and complete the square

$$ f(x) = a\left(x^2 + \frac{b}{a}x + \frac{c}{a}\right) \implies a\left(\left(x + \frac{b}{2a}\right)^2 - \frac{b^2}{4a^2} + \frac{c}{a}\right) \nonumber $$

Finally group the non $x$ terms in the bracket together

$$ a\left( \left(x + \frac{b}{2a}\right)^2 - \frac{b^2-4ac}{4a^2} \right) \nonumber $$

The graph of the $f(x)$ is thus obtained from $y=x^2$. This is shifted to the right by $-\frac{b}{2a}$, shifted downward by $\frac{b^2-4ac}{4a^2}$ and then scaled by $a$. Note that a negative value of $a$ will flip the graph upside down (reflection across the x axis) along with any scaling by $a$

Rational Functions

Rational function $f$ is defined as

$$ f(x) = \frac{p(x)}{q(x)} \nonumber $$

where both $p$ and $q$ are polynomials. We need to be careful to exclude points where $q(x)$ is zero from the domain of both $p$ and $q$.

An example of this is we ignore 0 from all real values in the domain like below and the range of $f$ is R

$$ f: \mathbb{R} \backslash \{0\} \rightarrow \mathbb{R} \space defined \space by \space f(x) = \frac{1}{x} \nonumber$$

Or any value that will make the denominator equal to zero. In this case, we ignore 1 from the domain and the range of $f$ is R

$$ f: \mathbb{R} \backslash \{1\} \rightarrow \mathbb{R} \space defined \space by \space f(x) = \frac{x^2}{x^3-1} \nonumber$$


Trignometric Functions

Before going into trignometric functions, we need to define radian and what it is. (This is a definition from the text book)

A radian is the angle subtended at the center of a circle by an arc which is the same length as the radius (subtended is a word I can never remember but it means opposite of something so in this context it is the angle at the center of of the circle which is opposite the arc, where the arc is a part of the circumference and this arc has the same length as the radius). If we have a unit circle where the radius is one, then the arc has a length of 1.

We know formula of circumference of circle is $2\pi r$ which then equates to 360 degrees. Thus on a unit circle:

$$ 2\pi \space radian = 360^\circ \nonumber $$

$$ \pi \space radian = 180^\circ \nonumber $$

$$ \pi/2 \space radian = 90^\circ \nonumber $$

And in general, using the below, we can convert between radian and degree.

$$ 1 \space radian = \frac{180^\circ}{\pi} \nonumber $$

For example,

$$ \frac{\pi}{3} \space radian \Rightarrow \frac{\pi}{3} . \frac{180^\circ}{\pi} = 60^\circ \nonumber $$

Trignometric functions are functions of angles which are measured in radians. If we see the below unit circle where angle $t$ and the point $(x,y)$ on the circumference of the circle. We can deduce that that $cos(t)= x$ and $sin(t) = y$. Using Pythagoras theorem, you can see that $$ cos^2(t) + sin^2(t) = 1 $$

Along with this, we also have other trignometric identities:

$$ tan(t) = \frac{sin(t)}{cos(t)};\space cot(x) = \frac{cos(t)}{sin(t)};\space sec(x) = \frac{1}{cos(t)};\space cosec(x) = \frac{1}{sin(t)} \nonumber$$

Here's the visualisation of the different trignometric functions.

A few observations here:

  • Because $cos(t)$ has a symmetic about the y-axis, it is an even function whilst $sin(t)$ is odd as you rotate the graph twice, you get back to the same graph $$cos(t) = cos(-t) $$ $$ sin(t) = -sin(-t)$$
  • $tan(t)$ is also an odd function $$ tan(t) = -tan(-t)$$
  • Worth noting that their reciprocal counterparts are also of the same classification i.e. $cosec(t)$ is odd, $sec(t)$ is even and $cot(t)$ is odd
  • Because these are periodic functions, they have a period of $2\pi$ meaning they repeat themselves every $2\pi$ distance (once again same with their reciprocal counterparts) $$ cos(t) = cos(2\pi+t) $$ $$ sin(t) = sin(2\pi+t) $$ $$ tan(t) = tan(2\pi+t) $$
  • $tan(t)$ repeats more often with period $\pi$ $$tan(t)=tan(t+\pi)$$
  • $cos(t)$ and $sin(t)$ flips with $\pi$ distance $$cos(t)=-cos(t+\pi) $$ $$ sin(t)=-sin(t+\pi)$$
  • At origin $(0,0)$, both $sin(t)$ and $tan(t)$ crosses it but $cos(t)$ does not. Interestingly, non of the reciprocal functions touches 0 since you cannot divide by 0 $$sin(0) = 0 ;\space cos(0) = 1;\space tan(0) = 1$$
  • At $\pi$ $$sin(\pi) = 0 ;\space cos(\pi) = -1;\space tan(\pi) = 0$$
  • $tan(t)$ and the reciprocal counterparts all have asymptotes. The asymptotes exists where the original function crosses the x axis.

Standard Trignometric Identities

There are a few formulas to prove here... First to tackle is the sum of two angles identities and then the difference of two angles identities. We can use these proven identities to derive the double angle, half angle, sum-to-product and product-to-sum identities.

The majority of the work to prove the identies is done by extending the original triangle by draw a rectangle around it and then deriving the vertices in term of $cos$ and $sin$. It is useful to draw this out on paper when going through it.

Sum of two angles formula

$$ sin(y+x) = sin(y)cos(x) + sin(x)cos(y) $$ $$ cos(y+x) = cos(y)cos(x) - sin(x)sin(y) $$

First we draw the triangle on the unit circle with the angle $(x+y)$. Note that point $C$ lies on the unit circle and $AC$ is the radius $1$.

We then extend the triangle by drawing a line parallel to $AB$ called $CD$. The angle $\angle ACD$ is also $(x+y)$. From $C$, we draw right angle triangle with angle $x$ and hypotenuse $AC$. We can then (very badly drawn) complete the rectangle like below

From the diagram above, we can see that: $$ CB = DF = DE + EF $$ $$ DC = FB = FA + AB $$

We can derive that (because the hypotenuse is 1 on the unit circle) $$ BC = sin(x+y) ,\space AB = cos(x+y) $$

On the otherside of the hypotenuse $AC$ for triangle $\triangle EDA $, we have $$ EA = sin(x) ,\space EC = cos(x) $$

We can also observe $EC$ is the hypotenuse for the right angle triangle $\triangle EDC $ since $\square FBCD$ is a rectangle with 4 square angles. Therefore... $$ cos(y) = \frac{DC}{EC},\space sin(y) = \frac{DE}{EC}$$

Since $ EC = cos(x) $, we can replace the $EC$ above and rearrange to get: $$ cos(y) = \frac{DC}{cos(x)} \Rightarrow DC = cos(x)cos(y) $$ $$ sin(y) = \frac{DE}{cos(x)} \Rightarrow DE = cos(x)sin(y) $$

Lastly, we can see that $\angle FEA$ is the same as angle $y$ or $\angle ECD$ since $$ \angle ECD + \angle DEC + \angle EDC = 180^o $$ $$ \angle FEA = 180^o - \angle CEA - \angle DEC \Rightarrow \angle FEA = y $$

Since we know the hypotenuse $EA = sin(x)$ and angle $\angle FEA = y$, we can deduce the vertices $EF$ and $FA$ as $$ cos(y) = \frac{EF}{EA} = \frac{EF}{sin(x)} \Rightarrow EF = cos(y)sin(x) $$ $$ sin(y) = \frac{FA}{EA} = \frac{FA}{sin(x)} \Rightarrow FA = sin(y)sin(x) $$

Ok, let's look at formulas we derived...

$$ AB = cos(x+y), \space FA = sin(y)sin(x) $$ $$ BC = sin(x+y) $$ $$ DC = cos(x)cos(y) $$ $$ DE = cos(x)sin(y), \space EF = cos(y)sin(x) $$

And finally putting it together, we can see the 2 formulas above:

$$ BC = DE + EF $$ $$ sin(x+y) = cos(x)sin(y) + cos(y)sin(x) $$ and $$ DC = FA + AB \Rightarrow cos(x)cos(y) = cos(x+y) + sin(y)sin(x) $$ $$ cos(x+y) = cos(x)cos(y) - sin(y)sin(x) $$

Difference of two angles formula

$$ sin(x-y) = sin(x)cos(y) - cos(x)sin(y) $$ $$ cos(x-y) = cos(x)cos(y) + sin(x)sin(y) $$

Like the sum of the two angles derivation, we also draw the triangle on the unit circle with the angles $x-y$. Note that point $C$ lies on the unit circle and $AC$ is the radius $1$.

To help visualise the angle $x-y$, we can extend another line from $A$ for angle $y$. This means that the total angle is $x$

If we extend this line until it forms a right angle triangle with hypotenuse $AC$ and then draw a rectangle around it. We will get the below.

Similar to above, $$AB = FD = FE + ED $$ $$ FA = DB = DC + BC $$

We can also derive for $\triangle ABC$ that $$ BC = sin(x-y),\space AB = cos(x-y) $$

On the otherside of the $AC$ hypotenuse for triangle $\triangle AEC$, we have $$ AE = cos(y),\space EC = sin(y) $$

Since now also have the hypotenuse $AE$ as well as angle $\angle AEF$ (same angle as $\angle EAB$ since they are supplementary angles made by parallel lines which adds up to $180^o \therefore$ $\angle AEF = x$), we can derive the other edges for the triangle $\triangle FAE$ $$ sin(x) = \frac{FA}{cos(y)} \Rightarrow FA = sin(x)cos(y) $$ $$ cos(x) = \frac{FE}{cos(y)} \Rightarrow FE = cos(x)cos(y) $$

The last triangle to look at is $\triangle EDC$. We know the hypotenuse is $EC$ which is $sin(y)$. Since we know that $\angle AEF = x \Rightarrow \angle ECD = x$ (should be fairly easy to prove why). With this we have $$ sin(x) = \frac{ED}{sin(y)} \Rightarrow ED = sin(x)sin(y) $$ $$ cos(x) = \frac{DC}{sin(y)} \Rightarrow DC = cos(x)sin(y) $$

Once again, let's assemble all the formulas we derived $$ AB = cos(x-y) $$ $$ FE = cos(x)cos(y),\space ED = sin(x)sin(y) $$ $$ FA = sin(x)cos(y) $$ $$ DC = cos(x)sin(y), \space BC = sin(x-y) $$

And then putting all together $$ AB = FE + ED $$ $$ cos(x-y) = cos(x)cos(y) + sin(x)sin(y) $$

and $$ FA = DC + BC \Rightarrow sin(x)cos(y) = cos(x)sin(y) + sin(x-y) $$ $$ sin(x-y) = sin(x)cos(y) - cos(x)sin(y) $$

Generalisation of sum and difference formulas & derivation of tan function

With the formulas above, we can see $$cos(x \pm y) = cos(x)cos(y) \mp sin(x)sin(y) $$ $$ sin(x \pm y) = sin(x)cos(y) \pm sin(y)cos(x) $$

Using the results above, we can also derive the $tan(x+y)$ and $tan(x-y)$. $$ tan(x \pm y) = \frac{sin(x \pm y)}{cos(x \pm y)} = \frac{sin(x)cos(y) \pm sin(y)cos(x)}{cos(x)cos(y) \mp sin(x)sin(y)} $$

If we divide both numerator and denominator with $\frac{1}{cos(x)cos(y)}$, the equation simplifies down significantly $$ tan(x \pm y) = \frac{\frac{sin(x)cos(y)}{cos(x)cos(y)} \pm \frac{sin(y)cos(x)}{cos(x)cos(y)}}{\frac{cos(x)cos(y)}{cos(x)cos(y)} \mp \frac{sin(x)sin(y)}{cos(x)cos(y)}} $$ $$ tan(x \pm y) = \frac{tan(x) \pm tan(y)}{1 \mp tan(x)tan(y)} $$

Double angle formula

Using the sum of two angles formula, we can derive the double angle formula. Instead of $sin(x+y)$ or $cos(x+y)$, we can replace the y.

For sin function $$ sin(x+y) \rightarrow sin(x+x) = sin(2x) $$ $$ \Rightarrow sin(x)cos(x) + sin(x)cos(x) $$ $$ \Rightarrow 2sin(x)cos(x) $$

For cos function $$ sin(x+y) \rightarrow cos(x+x) = cos(2x) $$ $$ \Rightarrow cos(x)cos(x) - sin(x)sin(x) $$ $$ \Rightarrow cos^2(x) - sin^2(x) $$

For tan function, we can use the 2 formula we just derived $$ tan(2x) = \frac{sin(2x)}{cos(2x)} = \frac{2sin(x)cos(x)}{cos^2(x) - sin^2(x)} $$

We can divide numerator and denominator with $cos^2(x)$ $$ tan(2x) = \frac{ \frac{2sin(x)cos(x)}{cos^2(x)} }{\frac{(cos^2(x) - sin^2(x))}{{cos^2(x)}}} $$ $$ \Rightarrow \frac{2tan(x)}{1-tan^2(x)} $$

You can also get the same formula for $tan$ using the $tan(x+y)$ formula we derived previously

Sine Rule

From the above functions, we also have the sine rule. This is given as

$$ \frac{sin A}{a} = \frac{sin B}{b} =\frac{sin C}{c} \nonumber $$

or

$$ \frac{a}{sin A} = \frac{b}{sin B} =\frac{c}{sin C} \nonumber $$

We can derive this by draw the line $h_B$ which create a right angle with AC from B like below.

Let write out all the bits we know

$$ sin A = \frac{h_B}{c}; \space sin C = \frac{h_B}{a} \Rightarrow cSinA = h_B; \space aSinC = h_B \nonumber$$

Since both are equal to $h_b$, we can write

$$ cSinA = aSinC \Rightarrow \frac{SinA}{a} = \frac{SinC}{c} \Rightarrow \frac{a}{sinA} = \frac{c}{sinC} \nonumber $$

In the case above, we drew line $h_B$ but we could also draw line $h_C$ from C and making a right angle with AB in a similar fashion like $h_B$ (or $h_A$ from A to BC). Going through the same logical reasoning, we can see that for $h_C$,

$$ bSinA = aSinB \Rightarrow \frac{SinB}{b} = \frac{SinA}{a} \Rightarrow \frac{a}{sinA} = \frac{b}{sinB} \nonumber $$

or for $h_A$,

$$ bSinC = cSinB \Rightarrow \frac{SinB}{b} = \frac{SinC}{c} \Rightarrow \frac{c}{sinC} = \frac{b}{sinB} \nonumber $$

From the above 3 results, we can very easily derive sine rule.

Cosine Rule

Next is cosine rule, this is given as:

$$ a^2 = b^2 + c^2 - 2bc \space cos A \nonumber $$

Using the same triangle above, if we set the distance $x$ to be from A to where $h_B$ meets B and therefore $h_B$ to C is $b-x$

We have the below:

$$ cosA = \frac{x}{c} \implies c \space cosA = x \nonumber $$

also applying the Pythagoras theorem to both inner right angle triangles, we get

$$ c^2 = h_B^2 + x^2 \space \And \space a^2 = h_B^2 + (b-x)^2 \nonumber $$

rearrange to remove the $h_B$, setting everything equals to $a^2$ and then expands the bracket

$$ a^2 = (b-x)^2 + (c^2 - x^2) \implies b^2-2bx+x^2 + c^2 - x^2 \nonumber $$

finally, cancelling the $x^2$ terms and replacing $x$ term with the $c \space cosA$

$$ a^2 = b^2-2bx + c^2 \implies b^2-2bc \space cosA + c^2 \implies b^2 + c^2 -2bc \space cosA \nonumber $$

You can do the same with side c and side b to get the results

$$ c^2 = b^2 + a^2 -2ba \space cosC \space \And \space b^2 = c^2 + a^2 -2ca \space cosB; \nonumber $$
